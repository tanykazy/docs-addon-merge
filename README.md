# docs-addon-merge

## 使い方

1. **データソースの準備**
   - Google スプレッドシートを用意し、1行目にヘッダー（列名）を入力します。2行目以降にデータを入力してください。

2. **テンプレートの作成**
   - Google ドキュメントを開きます。
   - メニューの「差し込み文書」>「データソースを選択」をクリックします。

3. **データソースの選択**
   - ダイアログが表示されるので、準備したスプレッドシートを選択します。

4. **フィールドの挿入**
   - サイドバーが表示されます。「シート」のドロップダウンから使用するシートを選択します。
   - 「フィールドコード」のリストから必要な項目を選び、コピーボタンをクリックするか、テキスト（例: `{{氏名}}`）をコピーして、ドキュメント内の差し込みたい場所に貼り付けます。
   - 必要なフィールドをすべて配置したら、「次へ」をクリックします。

5. **差し込みの実行**
   - データの一覧が表示されます。差し込みを行いたい行にチェックを入れます。
   - 右下の「完了」ボタンをクリックし、出力方法を選択します。
     - **1つのファイルに出力**: 選んだデータすべてを1つのドキュメントファイルに出力します。
     - **個別のファイルに出力**: 選んだデータごとに個別のドキュメントファイルを作成します。

6. **出力先の確認**
   - 出力が完了すると、ダイアログ内のリンクアイコンが有効になります。クリックすると作成されたファイルや保存先のフォルダを開くことができます。
   - 出力ファイルは、通常そのドキュメントと同じフォルダ、またはマイドライブ内に自動的に作成される「[差し込み文書]ドキュメント名」フォルダに保存されます。
